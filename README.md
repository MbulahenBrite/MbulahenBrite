## ✨I am curruntly enrolled on AWS re/Start

AWS re/Start is a **free, global training program** that helps unemployed or underemployed individuals gain the skills needed for entry-level cloud computing careers. It provides a combination of technical training, such as Linux, Python, and AWS core services, and professional skills like resume building and interview preparation.

### What the program includes:

| Category | Key Focus Areas |
| :--- | :--- |
| **Technical Skills** | Learners gain hands-on experience with fundamental IT and AWS cloud technologies through scenario-based exercises and labs. Topics include **Linux, networking, security, Python, and core AWS services.** |
| **Professional Skills** | The curriculum also focuses on "soft skills" crucial for a tech career, such as **communication, teamwork, time management, resume writing, and interview coaching.** |
| **Certification** | Graduates receive a voucher to take the **AWS Certified Cloud Practitioner** exam. |
